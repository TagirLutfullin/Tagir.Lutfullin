Ответы на вопросы:

1. Порядок приоритета стилей: inline > внутренние > внешние.
2. Inline стиль оптимален для одной страницы.
3. Внешние стили рекомендуются для больших проектов.
4. Недостатки способов: Inline - усложняет код, внутренние - средний приоритет, внешние - дополнительный HTTP-запрос.
